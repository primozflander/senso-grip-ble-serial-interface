senso-grip-ble-serial-interface
============

Interface between Arduino Nano BLE and serial communication needed for plotting data in Labview

Technologies:
-------------

Bluetooth Low Power

Prerequisites:
-------------
 
Arduino IDE 1.8.13,
Labview 2020

SETUP
-----

 1. unzip/clone code,
 2. open .ino file and upload it on Arduino Nano BLE board
